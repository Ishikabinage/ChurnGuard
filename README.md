# ChurnGuard 🛡️

A machine learning web app to predict customer churn for telecom companies.

## Tech Stack
- Python, Flask
- Scikit-learn, XGBoost
- SQLite
- HTML/CSS

## Models Used
- Logistic Regression
- Random Forest
- XGBoost

## How to Run
1. Clone the repo
2. Install dependencies: pip install -r requirements.txt
3. Run: python app/app.py
